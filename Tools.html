Assessment 3 | FlickDev Group

Tools

GitHub

FlickDev Group created a GitHub repository on the 28th October 2020, using GitHub Pages to host our website. Each team member has complete access to add, remove and edit when required.

Our website was created based on the HTML5 UP template ‘Editorial’, which was chosen for its simplicity and ease of navigation. We opted to use a template as wanted to focus our efforts on content, rather than design.

GitHub Audit Trail

We believe that the GitHub audit trail reflects both how tasks have been allocated for this project and personal working preferences. Some members, such as Matt, may have been more active at the start of the project as they uploaded the website template; other members, such as Natalie, may have been more active towards the end of the project when carrying out critical quality control and consistency checks and changes. Some team members have also chosen to work primarily in their local Git editors when constructing elements and committing large chunks or work to GitHub, while others have committed smaller changes more frequently.

Microsoft Teams

After setting up our group in Canvas, FlickDev Group has used Microsoft Teams extensively for communication, project management, assigning tasks, sharing ideas, uploading, editing and polishing work and reports, and provide feedback. We used Microsoft Teams to host weekly or bi-weekly video meetings through the project, recording each meeting and the relevant agenda and action notes.

Figma

In order to develop and deliver a working prototype of the FlickDeck app, FlickDev Group chose the interface design application Figma.

Figma, which can be accessed either through a web interface or desktop app, allows the FlickDev Group to collaboratively design and create wireframes and working prototypes of the Flick Deck app. Figma is highly focused on live collaborative workflows, and offers the ability to edit files together in real-time or view the action in “observation mode”.

Canva
